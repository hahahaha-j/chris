Introduction

Chris Cross is the perfect bot for your servers, especially for users who feel lonely in their servers.

About this Bot

This bot links multiple channels across multiple servers so that users can talk to each other powered by Discord

Technology

The bot is bulit with python using the discord.py and guilded.py libraries.

The bot uses it's own internal API that allows it to talk back and forth between your Guilded server and Discord server, with less than 0.5 seconds of latency!

Getting Started

Getting started with the bot is easy. Simply run the /request command, add in the channel ID, make sure the bot has the correct permissions, and we will notify you when the channel is accepted.

Commands

All commands

  • /request
    • Description: Request access to use the bot.
    Usage: /request channel:
    Example: /request channel:1234567890
    Note: Please make sure that the bot has the correct permissions to view the channel.
  • /what
    • Description: Get more info about the bot and what it does
    Usage: /what
    Example: /what
    Note: Only you can see the message.
  • report message to moderators
    • Description: If you see a message that violates Chris's ToS, you can use this to report the message to moderators.
    Usage: Click message - Click Apps - Click "report message to moderators"
    Example: Fuck you - Report
    Note: Spam of this system can result in a suspension / elimination of you or your server's access to Chris.
  • Autobot Reminder
    • Description: Since Discord's webhook messages have a little BOT tag next to the name, we often get people asking "Why is everyone a bot?" An AI algorithm will scan your message, and if it thinks you are asking about the "bots", it will block your message and remind you that these are users.
    Usage: why bots < character name>
    Example : why are you bots?
    Note: Sometimes it is not accurate.
